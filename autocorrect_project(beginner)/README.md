# Autocorrect Keyboard System

## 📌 Description
This project is an NLP-based autocorrect system built using Python. It detects spelling mistakes in words and sentences and provides correct suggestions.

## 🚀 Features
- Word-level autocorrection
- Sentence correction
- Suggestions for incorrect words
- User-friendly GUI interface

## 🛠️ Technologies Used
- Python
- pyspellchecker
- tkinter

## ▶️ How to Run

1. Install required library:
   pip install -r requirements.txt

2. Run the GUI application:
   python gui.py

## 🧪 Example

Input:
I havv goood speling

Output:
I have good spelling

## 📂 Project Structure
- autocorrect.py → core logic
- gui.py → GUI application
- requirements.txt → dependencies
- README.md → project documentation

## 🎯 Conclusion
This project demonstrates a basic Natural Language Processing (NLP) application that improves text input accuracy using autocorrection techniques.
